# ok-IN 开发计划

## 项目概述
基于 ok-script 框架开发《无限暖暖》自动化程序 ok-IN，支持后台运行。
1.0 目标：一键日常。

## 技术栈
- Python 3.12
- ok-script >= 1.0.148
- OpenCV (模板匹配)
- OnnxOCR (文字识别)
- PySide6 (GUI)

## 游戏技术信息
- 主进程: X6Game-Win64-Shipping.exe
- 窗口类: UnrealWindow
- 交互模式: PostMessage (支持后台)
- 反作弊: Anti-Cheat Expert (ACE)
- 推荐分辨率: 1920x1080 (16:9)

## 积分消耗控制策略
- 每日上限: 1300 积分
- 代码编写与搜索: ~600-800/天
- 文件读写与测试: ~200-300/天
- 预留缓冲: ~200-300/天
- 大量搜索/截图标注日适当降低代码产出量

## 风险与应对
1. **游戏更新导致UI变化**: 需要重新截图标注模板 → 及时跟踪游戏更新公告
2. **反作弊升级**: ok-script 仅模拟用户输入，风险较低 → 关注社区反馈
3. **ACE 截帧保护**: 使用 WGC/BitBlt 截图方式，不使用 hook → 保持与鸣潮方案一致
4. **模板匹配精度不足**: 某些界面相似度高 → 结合 OCR 辅助判断
