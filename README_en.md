# ok-IN

An image-recognition-based automation tool for Infinity Nikki, with background mode support.

Built with the [ok-script](https://github.com/ok-oldking/ok-script) framework.

## ✨ Features

- **Background Mode**: Supports automation while the game window is in the background.
- **One-Click Daily**: Automatically completes daily routines (Daily Wishes, Realms, Pear-Pal dispatch, mail claiming).
- **Auto Login**: Handles the login process automatically.
- **Skip Dialog**: Skips lengthy story dialogues.

## 🖥️ Requirements

- **OS**: Windows
- **Game Resolution**: 1920x1080 or higher (16:9 aspect ratio recommended)
- **Game Language**: Simplified Chinese / English
- **Python**: 3.12+

## 🚀 Installation

### Option 1: Installer (Recommended)

Download the latest installer from [GitHub Releases](https://github.com/ok-oldking/ok-IN/releases).

### Option 2: From Source

1. Ensure **Python 3.12+** is installed.

2. Clone the repository:
```bash
git clone https://github.com/ok-oldking/ok-IN.git
cd ok-IN
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run:
```bash
# Release mode
python main.py

# Debug mode (more verbose logging)
python main_debug.py
```

## 📖 Usage

### Before You Start

- **Graphics Settings**: Disable all custom UI modifications, filters, and sharpening effects. Use default brightness.
- **Resolution**: 1920x1080 or higher recommended.
- **Key Bindings**: Use default in-game key bindings.
- **Window**: Game window can be in the background but must not be minimized. Do not let your screen lock or turn off.
